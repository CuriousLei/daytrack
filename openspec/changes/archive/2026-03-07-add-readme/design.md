## Context

这是一个纯文档补充任务，不涉及代码修改。项目现有功能已基本实现，需要为用户提供清晰的使用说明。

## Goals / Non-Goals

**Goals:**
- 提供完整的项目介绍
- 说明安装和配置步骤
- 详细列出所有可用命令及其用法
- 提供常见问题解答

**Non-Goals:**
- 不修改任何代码
- 不改变现有功能
- 不添加新的依赖

## Decisions

### 1. README 结构
- 项目简介
- 功能特性
- 安装说明
- 快速开始
- 命令参考
- 配置说明
- 常见问题

### 2. 风格选择
- 使用 Markdown 格式
- 简洁明了的示例代码
- 中英文双语（主要中文，代码示例英文）

## Risks / Trade-offs

| 风险 | 缓解措施 |
|------|---------|
| 文档与代码不一致 | 先基于现有代码功能编写，后续随功能更新同步维护 |
