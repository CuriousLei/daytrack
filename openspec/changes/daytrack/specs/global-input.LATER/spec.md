# Global Input 规范

## 概述

类似 Alfred 的全局悬浮输入框，用于快速输入临时想到的、系统无法自动采集的工作项。

## 功能需求

### 1. 全局快捷键
- 可配置的全局快捷键（默认建议 Cmd/Ctrl + Shift + Space）
- 支持在任何应用中唤起
- 快捷键冲突时提供提示和重新配置

### 2. 悬浮输入界面
- 极简的输入框设计
- 显示在屏幕中央或可配置位置
- 支持快速输入，自动保存
- 输入后按回车保存，按 Esc 取消/关闭

### 3. 输入功能
- 支持纯文本输入
- 支持 Markdown（可选）
- 输入后立即保存到当天的工作痕迹
- 可以选择添加标签/分类（可选）

### 4. 历史和快速访问
- 显示最近输入的历史（可选）
- 支持快速选择和重复使用之前的输入
- 支持编辑之前的输入

### 5. 跨平台支持
- macOS：支持
- Linux：支持
- Windows：支持（后续可选）

## 非功能需求

- 响应速度：唤起和输入应该快速流畅
- 低资源占用：后台运行时占用资源少
- 可靠性：输入应该可靠保存，不丢失数据

## 技术栈建议

- Electron（跨平台）
- 或者 Tauri（更轻量，后续可考虑）
- macOS：也可以考虑用 native 方案 + SwiftUI

## 扩展点

- 未来可以支持语音输入
- 未来可以支持快速命令（类似 Alfred workflows）
- 未来可以支持 AI 补全和建议
- 未来可以支持截图/图片附件
