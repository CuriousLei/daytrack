# Web UI 规范

## 概述

极简现代风格的网页界面，用于查看和编辑每日日报。

## 功能需求

### 1. 技术栈
- **前端框架**：Preact
- **构建工具**：Vite
- **样式方案**：Tailwind CSS
- **组件库**：shadcn/ui（基于 Tailwind）
- **状态管理**：Preact 内置 state + context（第一期）

### 2. 页面结构

```
/                    # 首页 → 重定向到今天
/today              # 今天的日报
/report/:date      # 某一天的日报
/reports            # 日报列表
/settings           # 设置页面
```

### 3. 组件设计

```
components/
├── Layout.tsx         # 布局（侧边栏、头部）
├── DailyReport.tsx    # 日报详情页
├── ReportList.tsx     # 日报列表
├── ReportEditor.tsx   # 日报编辑器
├── TraceList.tsx      # 工作痕迹列表
├── Settings.tsx       # 设置页面
└── ui/                # shadcn/ui 组件
    ├── button.tsx
    ├── card.tsx
    ├── input.tsx
    ├── ...
```

### 4. 主要功能

#### 日报详情页 (/today, /report/:date)
- 显示日报内容
- 区分 AI 生成内容和用户编辑内容
- 提供编辑按钮
- 显示当天的原始工作痕迹（可选展开/收起）
- 提供"重新生成"按钮
- 前一天/后一天快捷导航

#### 日报列表页 (/reports)
- 按月份分组显示
- 显示每天的摘要（前几句话）
- 支持按月份/年份筛选
- 快速跳转到某一天

#### 设置页 (/settings)
- 配置大模型（API Key、模型选择）
- 配置 git 仓库路径
- 选择日报风格（默认/简洁/技术）
- 自定义 prompt 模板（高级选项）

#### 编辑功能
- 实时编辑日报内容
- Markdown 预览（可选）
- 保存编辑历史（可选）
- 编辑后可以选择是否重新生成

### 5. UI 设计原则
- **极简现代风格**：简洁、清爽、视觉层次清晰
- **响应式设计**：支持桌面和移动设备
- **深色/浅色模式**：支持切换
- **加载状态**：友好的加载和错误提示

## 非功能需求

- 加载速度：页面加载和交互应该流畅
- 可访问性：基本的可访问性支持
- 本地优先：所有数据在本地，不需要网络（除了大模型调用）

## 技术选型

| 技术 | 选型 | 理由 |
|------|------|------|
| 框架 | Preact | 轻量，React API 兼容 |
| 构建工具 | Vite | 开发体验好，构建快 |
| 样式 | Tailwind CSS | 实用优先，快速开发 |
| 组件库 | shadcn/ui | 极简现代，基于 Tailwind |
